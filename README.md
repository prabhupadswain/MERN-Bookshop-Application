# MERN-Bookshop-Application

 It is a book ordering application developed in MERN Stack. It has features like retrieving information about books & manipulating its data along with a feature of making an order for purchasing of books & retrieve the information about the past orders related to book. 

-> This application in its initial version is intended to emulate the working process of how a bookshop or a bookstrore operates in real-world. 
Customers come to view information about books, make an order for purchasing a book, view their past orders with the shop etc.

-> The application can be used by the administrator or the bookshop owner to operate how its book inventory, bookshop can operate through software solutions.
